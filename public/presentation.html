01 — Problematique

Le quotidien au Maroc :
trop d'apps, pas de solution unifiee

  • Aucun outil centralise pour gerer ses documents (CIN, permis, passeport...)
  • Impossible de comparer facilement les prix entre supermarches
  • Pas de suivi budgetaire adapte au contexte marocain (DH, categories locales)
  • Trouver un prestataire fiable (plombier, electricien) reste difficile
  • Comparer les prix de transport (taxi, train, bus) necessite plusieurs sites
  • Aucune application tout-en-un pensee pour le Maroc

04 — Technologies

Stack technique moderne

Frontend

  • Next.js 16 (App Router) + React 19
  • TypeScript pour la surete du code
  • Tailwind CSS 4 pour le design
  • Framer Motion pour les animations
  • Axios + Cookie-based auth
  • Deploye sur Vercel (CDN mondial)

Backend

  • Laravel 11 + PHP 8.3
  • API RESTful — 38+ endpoints
  • MySQL — 24 tables normalisees
  • Laravel Sanctum (auth JWT)
  • Groq API (IA Llama 3.3 70B)
  • Deploye sur Railway (cloud)

06 — Base de donnees

Architecture des donnees

24 tables MySQL normalisees

  • users — comptes utilisateurs
  • documents — CIN, permis, passeport
  • transactions — revenus et depenses
  • transport_routes — itineraires
  • transport_options — prix par type
  • service_categories — types de services
  • service_providers — prestataires
  • reviews — avis et notes

Module prix & promotions

  • product_categories — categories produits
  • products — catalogue produits
  • stores — BIM, Marjane, Carrefour
  • product_prices — prix par magasin
  • promotions — promos actives
  • chat_messages — historique IA
  • personal_access_tokens — auth JWT
  • + autres tables — migrations, cache...
Relations 1:N Foreign keys Indexes Seeders

07 — Securite & Deploiement

Production-ready & securise

🔒

Securite

  • Authentification JWT (Laravel Sanctum)
  • CORS strict — origines specifiques
  • Headers securite (HSTS, X-Frame-Options)
  • Rate limiting — 60 requetes/minute
  • Validation des entrees cote serveur
  • HTTPS obligatoire en production
🚀

Deploiement

  • Frontend : Vercel — CDN mondial, SSL auto
  • Backend : Railway — cloud scalable
  • BDD : MySQL heberge sur Railway
  • SEO : Open Graph, Twitter Cards, Meta tags
  • PWA manifest pour installation mobile
  • URL publique : hayati.cfd
